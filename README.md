# HOUSING-2.0
This repository contains the R Markdown code for the main simulation results in: Kökbudak, A.B. and Nikolaidi, M. (2026). 'Green securitisation and housing decarbonisation in an ecological stock-flow consistent model'.
To run this code: (1) install R and RStudio; (2) click on the 'Code' button and download and extract the ZIP folder; (3) open the .Rmd file in RStudio and run the code by clicking on the 'run' button.
The code generates Figures 2 and A.1 in the paper, as well as the parameter and variable tables in the appendix. Before running the code, install the required R packages using the following commands: install.packages(c("ggplot2", "reshape2", "ggthemes", "showtext","sysfonts", "showtextdb", "xtable"))
For more information about the DEFINE-HOUSING model, see here. For any questions or comments, please write to: a.kokbudak@greenwich.ac.uk or m.nikolaidi@greenwich.ac.uk.
